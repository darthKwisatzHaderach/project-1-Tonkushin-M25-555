# Лабиринт сокровищ

Консольная текстовая приключенческая игра на чистом Python. Ходите по комнатам, подбирайте/используйте предметы, решайте загадки и откройте сундук с сокровищем.

## Быстрый старт

1) Установите Poetry 1.8.4+ и включите локальное venv:
```
poetry config virtualenvs.in-project true
```
2) Инициализация окружения и установка:
```
make install
```
3) Запуск через entrypoint-скрипт:
```
make project
```

## Полезные команды
- Сборка пакета:
```
make build
```
- Псевдопубликация (dry run):
```
make publish
```
- Установка собранного колеса в систему:
```
make package-install
```
- Линтинг Ruff:
```
make lint
```

## Команды в игре
- `help` — показать доступные команды
- `look` — осмотреть текущую комнату
- `inventory` — показать инвентарь
- `go <north|south|east|west>` — перейти по направлению
- `north|south|east|west` — короткая форма движения
- `take <item>` — взять предмет из комнаты
- `use <item>` — использовать предмет из инвентаря
- `solve` — попытаться решить загадку (в сокровищнице — открыть сундук)
- `quit` — выйти из игры

Альтернативный запуск без Makefile:
```
poetry run python -m labyrinth_game.main
```

## Демонстрация (Asciinema)
- Установите asciinema и запишите сессию с запуском и победой:
```
brew install asciinema
asciinema rec
# запустите: make project → пройдите до победы → Ctrl+D для завершения
```
- Вставьте ссылку на запись сюда:
```
https://asciinema.org/a/REPLACE_WITH_YOUR_CAST
```
